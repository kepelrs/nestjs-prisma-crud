Quickstart

info

This guide assumes you have already setup NestJS and Prisma. See Getting Started if you haven't.


  1. Install nestjs-prisma-crud and nestjs-prisma-crud-schematics:

    npm i nestjs-prisma-crud --savenpm i -g nestjs-prisma-crud-schematics

  2. Generate the entire crud module with the following command (replace post with some your table's name from your Prisma schema):

    nest g -c nestjs-prisma-crud-schematics crud-resource post

  3. Start the server:

    npm run start:dev

  4. Verify installation by navigating to http://localhost:3000/post
